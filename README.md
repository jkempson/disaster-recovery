# Disaster Recovery Documentation

This repository contains disaster recovery procedures and guides for various infrastructure components.

## Contents

- **[JUICEFS-DISASTER-RECOVERY.md](JUICEFS-DISASTER-RECOVERY.md)** - Complete disaster recovery guide for JuiceFS filesystem with Backblaze B2 storage

## Usage

These guides are designed to be accessible during disaster scenarios. Clone this repository or bookmark specific guides for quick access during emergencies.

```bash
git clone https://github.com/jkempson/disaster-recovery.git
```

## Security Note

All credentials and sensitive information have been redacted from these guides. Replace placeholders (e.g., `<YOUR_B2_KEY_ID>`) with your actual values from your secure password manager.
